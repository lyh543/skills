# Save Chat History Skill

This skill organizes a raw chat transcript into a cleaned Q&A markdown file.

## Model Recommendation

This task is **purely mechanical** (reformatting and summarizing text), so it does not require a capable or expensive model.

Recommended: use a **cheap reasoning model with low reasoning effort**, for example:

- `GPT 5.4-mini` (low reasoning)

High-end models are wasteful here and will not produce meaningfully better output.
